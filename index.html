About Me

I'm a final-year Software Engineering student at the University of Zambia, passionate about building innovative web and mobile solutions that serve local Zambian communities. My focus areas include student accommodation platforms, agritech applications for smallholder farmers, and secure fintech escrow systems. Through CSC4035 Web Programming, I've mastered responsive design with Flexbox and CSS Grid, creating pixel-perfect layouts that work across all devices. I thrive in collaborative Git environments and enjoy tackling complex domain problems with clean, scalable code. My technical skills span full-stack development (HTML/CSS/JS, React, Node.js, Python/Django) and I'm particularly excited about leveraging technology to solve real-world challenges in Zambia's growing tech ecosystem.I am constantly exploring new technologies and frameworks to improve my development skills. I enjoy working on projects that create real impact in society, especially solutions that help students, farmers, and small businesses. My goal is to become a professional software engineer contributing to Africa’s digital transformation.
